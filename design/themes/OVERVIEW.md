# RimeConfig 0.2.0 · 主题皮肤设计交付概览

> 交付性质：设计规格（仅设计，不接入功能模块）
> 目标：将 3 种视觉风格做成**可切换的完整皮肤包**（配色 + 字体 + 装饰）

## 交付物清单

| 文件 | 内容 |
|---|---|
| `TOKENS.md` | 共享基础 + A/B/C 三套令牌对照表（色值/字体/语义色）+ WCAG AA 对比 + QSS 落地与设置页切换建议 |
| `application_light.qss` | 主题 A — Fluent Light QSS 骨架 |
| `application_dark.qss` | 主题 B — Dark Pro QSS 骨架 |
| `application_ink.qss` | 主题 C — Ink Scholar QSS 骨架（含专属装饰注入说明） |
| `settings_theme_switcher.png` | 设置页「主题」三选项卡片式切换界面稿 |
| `settings_theme_switcher.html` | 上述界面稿源文件 |

## 关键设计决策

1. **完整皮肤包**：三套皮肤共用同一布局/组件系统，仅换令牌值；C 的宋体标题、朱砂印章、左侧渐变条随皮肤一并切换，不丢失水墨气质。
2. **共享基础锁定**：窗口 1180×760、标题栏 44、导航 208、控件高 36、表格行高 44、圆角 8/10/20、间距 4 基准——切换时窗口零抖动。
3. **QSS 落地**：推荐「单模板 + 占位符替换」（theme.py 内置三套令牌字典做字符串替换），与现有 `theme.py` 机制一致；亦可三份独立 QSS。
4. **设置页切换**：「主题」下拉由 浅色/深色 扩展为 浅色(A)/深色(B)/水墨(C)，内部值 `light`/`dark`/`ink`；沿用 `themeChanged` 即时切换并持久化（不重建对象、不丢表单、不部署）。
5. **可访问性**：三套主/次文字对比均 ≥ 4.5:1（AA），焦点 2px 实线 outline。
