## 已完成：系统词典候选提示与权重建议

### 目标

在采集窗口中检测系统词典是否已存在同文本、同编码的候选；若命中，展示词典来源、参考原始权重与本工具参考优先级，并允许用户采用高于参考值的自定义权重。

### 实现

- 不在采集时扫描 Rime 编译词库或大 YAML 词库。
- 程序启动和采集窗口打开时，在后台解析当前 Frost 静态词典源文件，建立工具自身的 SQLite 索引。
- 索引键为文本和规范化编码，记录来源、原始权重、本工具参考优先级及词典文件签名。
- 检测到词典文件签名变化后重建索引；查询阶段只做精确索引查询。
- 索引未完成时，采集功能照常可用，并显示“系统词典索引准备中”。

### 当前配置依据

- 主系统词典：`rime_frost`，本工具参考优先级 `1.2`。
- 英文词典：`melt_eng`，本工具参考优先级 `1.1`。
- 中英混合用户词典：`cn_en`，本工具参考优先级 `0.5`。
- 本工具管理的自定义短语：`custom_phrase`，本工具参考优先级 `99`，通常已具有较高候选优先级。

### 边界

- 静态索引可准确提示“系统词典已有同词同码”及其原始权重。
- 不承诺完全复现输入法实时第一候选，因为最终排序还会受过滤器、学习记录和翻译器质量影响。

## 已完成：托盘菜单语义与入口

### 现状与问题

- “设置”动作当前只切换隐藏窗口的选项卡，未显示或激活主窗口，因此从托盘点击时看似无效。
- “一键部署”实际是立即调用 `WeaselDeployer.exe /deploy`，用于重新编译并加载 Rime 配置；它不是持久化状态。

### 实现

- “设置”执行显示主窗口、激活窗口、切换到设置页。
- 将“一键部署”改名为“立即重新部署”，保持普通命令按钮，不使用勾选状态。
- 托盘中已增加独立的可勾选项“保存后自动部署”，映射现有 `auto_deploy` 设置并即时持久化。
- 当处于沙盒模式或未找到部署器时，禁用“立即重新部署”并说明原因；自动部署开关仍可保留用户偏好但不触发真实部署。

## 已完成：词库表格批量选择与批量操作

### 目标

- 提供明确的“批量选择”按钮，进入批量模式后显示选择状态。
- 支持点击行、点击选择框，以及拖拽框选连续或多个词条。
- 对已选词条提供批量删除和批量调整分组。

### 实现

- 保留现有选择列作为批量模式的可见复选框；行点击在批量模式下切换该行选中状态。
- 允许使用 Ctrl/Shift 进行多选和连续范围选择，并将视图选择状态同步至批量勾选集合。
- 工具栏提供“全选 / 取消全选”“批量删除”“调整分组”操作。
- 批量删除沿用一次备份、一次保存、一次后台部署及结果通知。
- 批量调整分组弹出分组选择窗口；支持选择现有分组或“未分组”，并在一次保存后刷新表格和侧栏统计。
