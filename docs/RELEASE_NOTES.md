# 本地发布记录

## v0.4.2 - 2026-07-14

### Rime 候选预览

- 采集、单条编辑和多编码编辑窗口均提供独立 Rime 会话的前五候选预览。
- 候选条采用“候选：”与“权重：”双行布局；当前采集文本使用主题强调色标记。
- 系统词典索引支持空格分隔拼音与连续输入编码的规范化匹配，可显示如“完整 / 测试”等系统词典原始权重。
- 候选预览服务在程序启动后后台预热，并在部署成功后自动刷新；空闲后自动退出。
- 候选读取、完成刷新前后预留固定区域，避免采集和编辑窗口内容发生跳动或压缩。

### 说明与验证

- 系统词典提示中的“本工具参考优先级”仅用于整理来源提示，不参与 Rime 实际候选排序。
- 自动化测试：`116 passed`。
- 已知测试环境提示：`.pytest_cache` 无写权限会产生一条警告，不影响程序功能。

## v0.4.1 - 2026-07-13

本版本为本地 Windows 发布包，不上传 GitHub。

### 采集与托盘

- 快速采集窗口会在后台建立系统词典索引；命中同文本同编码时显示来源、翻译器质量与参考权重，可直接采用建议权重。
- 托盘“设置”会显示主窗口并定位设置页；“一键部署”更名为“立即重新部署”，并增加“保存后自动部署”开关。
- Win32 热键采集的剪贴板轮询改用单调时钟并扩展等待窗口，覆盖复制降级时的延迟结果。

### 批量操作与界面稳定性

- 词库批量模式支持行点击、左侧方框和鼠标拖拽连续框选；对勾由程序直接绘制，不依赖外部图标资源。
- 设置页所有复选框改用自绘白色对勾，浅色、深色和水墨主题下均可明确辨认选中状态。
- 修复无界面自动化测试中延时提示框可能访问已销毁页面的问题。

### 验证与产物

- 自动化测试：`116 passed`。
- 已知测试环境提示：`.pytest_cache` 无写权限会产生一条警告，不影响程序功能。
- 本地可执行文件：`dist/RimeConfig.exe`。

## v0.4.0 - 2026-07-13

本版本为本地 Windows 发布包，不上传 GitHub。

### 词库与编辑

- 支持立即备份、定期备份、保留份数和自动清理旧备份。
- 备份和恢复覆盖受管词库、输入方案、符号表及 `pinyin_display.ini`。
- 采集和编辑窗口支持一个文本维护多个编码，含候选编码的新增、移除和权重调整。
- 英文词条提供小写、大写和原样编码建议；可选择将输出文本转为大写而保持编码小写。
- 多编码删除支持批量勾选，收藏、编辑、删除均保留窗口状态提示与右下角通知。

### 热键、通知与运行

- 修复热键未选中文本时误采集程序工作目录的问题，Win32 与 keyboard 后端均覆盖路径变体。
- 热键采集、自动部署、检查更新和下载更新均移出界面主路径。
- 新增应用内托盘通知补偿，通知展示文本和编码并可手动关闭。
- 新增 Windows 单实例锁，重复启动不会再打开第二个窗口。

### 界面与性能

- 备份设置集中在“备份与恢复”；所有下拉框和数值框均需点击后才响应滚轮。
- 完成浅色、深色和水墨主题的统一语义颜色。
- 词库搜索增加防抖，分组和筛选避免重复整库排序。
- 拼音显示推导加入内存缓存；符号表批量刷新减少布局开销。
- 修正扩展词库统计：英文输入文件 4 个，中英混合用户词库 7 个，不再把脚本或源码计入。
- 日志采用 1MB 轮转并保留 3 份，避免长期运行持续增长。

### 验证与产物

- 自动化测试：`110 passed`。
- 已知测试环境提示：`.pytest_cache` 无写权限会产生一条警告，不影响程序功能。
- 本地可执行文件：`dist/RimeConfig.exe`。
