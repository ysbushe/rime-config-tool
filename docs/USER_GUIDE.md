# RIME 配置小工具用户说明

## 开始使用

首次打开后，在“设置”中确认 Rime 用户目录。程序会自动寻找常见的小狼毫目录；未识别时可手动选择。目录有效后，可在同一页面检查受管文件并执行“立即重新部署”。

## 词库管理

词库管理页用于维护 `custom_phrase.txt`。可搜索文本或编码、按分组筛选、调整权重和删除项目。双击文本或编码可打开编辑窗口；同一文本可保存多个编码。

“批量选择”开启后，可点击行、勾选框或拖拽连续选择，再执行批量删除或调整分组。完成操作后程序会退出批量选择状态。

## 采集与候选预览

使用设置的全局热键采集当前已选中的文本。采集窗口会给出全拼及简拼建议，也会显示独立 Rime 会话的前五项候选和可读取的权重信息。候选预览仅用于辅助判断，不会改变正在使用的输入法。

纯英文文本可选择小写、大写或原样编码；英文短语的空格和连字符会自动移除。

## 分组、重码与历史

分组侧栏显示各组条目数。点击“显示重码字符/文本”可按同文本多编码或同编码多文本查看；双击项目后可集中编辑，点击“保存全部”才写入词库。

“修改历史”可查看词库维护记录；词库检查可发现空值、重复项、权重或分组问题。

## 备份与恢复

每次关键保存前会创建受管文件备份。设置页可立即备份、启用定期备份、设定保留份数和自动清理，并直接打开备份文件夹。

恢复时先选择受管文件和对应备份版本，再点击“恢复所选备份”。恢复前程序会先备份当前文件。

## 托盘、部署与更新

关闭窗口后程序仍驻留系统托盘。托盘菜单可显示主窗口、立即重新部署、切换保存后自动部署和开机自启。

“检查更新”会自动识别当前运行的是单文件 `RimeConfig.exe` 还是目录式便携包：前者只下载并替换 EXE，后者只下载 `RimeConfig-portable.zip` 并整体替换程序目录。更新失败会自动恢复旧版本。

## 主题与数据位置

程序提供浅色、深色和水墨三种主题。程序设置、历史和缓存保存在当前 Windows 用户的应用配置目录；发布版默认备份目录位于“用户文档\RIME 配置小工具\Backups”。
